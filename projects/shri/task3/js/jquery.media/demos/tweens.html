Tweens:

  • <Enter> Start/end a capture
  • < Move the cursor -0.2 seconds
  • > Move the cursor +0.2 seconds
  • <space> Play/pause the video
  • click Create a new point in the capturing tween
  • alt + click Deletes the last point in the capturing tween
  • shift + click Create a new point in the capturing tween and define a new status
